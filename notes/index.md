## Getting Started

1. Place research material in inbox/ with provenance metadata
2. Run `/reduce inbox/[filename]` to extract atomic notes
3. Run `/reflect [note title]` to find connections
4. Run `/verify [note title]` to quality-check
5. Add the note to a topic map or create one if 5+ related notes exist
